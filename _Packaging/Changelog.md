# Changelog
All notable changes to this project will be documented in this file.

## 2.0 - 2026-01-12
### Added
- Created complete localization system with 11 language files (enUS, deDE, esES, esMX, frFR, itIT, koKR, ptBR, ruRU, zhCN, zhTW)

### Changed
- Standardized keyboard shortcut format to "Ctrl + X" (with "Strg + X" for German) across all languages
- Renamed main files from Krowi_PopupDialog-1.0.* to Krowi_PopupDialog.* for cleaner structure
- Updated TOC file with expanded interface version support (12.0.0, 11.0.7, 5.0.5, and Classic versions)
- Modernized TOC metadata format (X-Slug, X-CurseForge-ID, X-Wago-ID)
- Updated copyright headers to reference LICENSE file

## 1.0.3 - 2026-01-01
### Added
- New numeric input dialog functionality with `ShowNumericInput` function
- Support for customizable min/max values, default value, and callback in numeric input dialog

## 1.0.2 - 2025-12-07
### Changed
- Codebase for upload
